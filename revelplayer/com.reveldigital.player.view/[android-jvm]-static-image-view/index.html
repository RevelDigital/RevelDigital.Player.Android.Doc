StaticImageView

open class StaticImageView : AppCompatImageView, RevelControl, DownloadFileTask2.OnCompleted

A custom androidx.appcompat.widget.AppCompatImageView for displaying static images, including raster formats (like PNG, JPG) and SVG vector graphics.

This view can load images from local file paths or remote URLs. For remote URLs, it utilizes DownloadFileTask2 to fetch and cache the image. It supports different ScaleType options for displaying the image, with FIT_XY as the default. SVG images are rendered using the SVGHelper library and are typically scaled with FIT_CENTER.

It implements RevelControl for integration into the RevelDigital player system, allowing it to be managed as a content module, and DownloadFileTask2.OnCompleted to handle the completion of asynchronous image downloads.
