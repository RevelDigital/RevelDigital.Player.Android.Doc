QRCodeView

open class QRCodeView : AppCompatImageView, RevelControl, DownloadFileTask2.OnCompleted

A custom androidx.appcompat.widget.AppCompatImageView that displays a QR Code.

This view generates a QR code image based on provided text and styling options (foreground and background colors). It fetches the QR code image from a remote service and displays it. The generation and display are typically triggered during initialization and can be updated by changing the text content.

It implements RevelControl for integration into the RevelDigital player system, allowing it to be managed as a content module. It also uses DownloadFileTask2 for asynchronous image downloading.
